# 🌌 Meu Universo

Projeto interativo inspirado no espaço, com animações, áudio e efeitos visuais.

## 🚀 Funcionalidades
- Sistema solar animado
- Parallax com profundidade
- Efeito de viagem espacial (warp)
- Nave espacial dinâmica
- Sons interativos

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript

## 🎮 Como usar
Clique em "Explorar galáxia" para iniciar a experiência.
